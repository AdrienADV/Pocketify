Support

Help and contact information for Pocketify.

Get help

For bugs, questions, or feature requests, open an issue in the Pocketify GitHub repository.

Before reporting an issue

  • Check that your Coolify instance URL is correct.
  • Make sure your Coolify API token is valid and has the required permissions.
  • Include your app version, device model, operating system version, and the steps to reproduce the issue.
